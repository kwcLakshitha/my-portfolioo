Smart School System

The Smart School System is Lakshitha Withanage's
first-semester final project, designed to streamline school
management.It includes features like role-based access,
student and teacher management,attendance tracking,
grade management,and timetable creation.
This system efficiently handles both academic
and administrative tasks, improving school operations.

TIC TAC TOY Game

A console-based Tic-Tac-Toe game developed in Java,
both two-player mode and an AI player.
In the AI mode,the computer makes optimal moves to
challenge the player.The game checks for a winner
after each move and announces the result.
This project showcases my skills in implementing game
logic,user interaction, and AI functionality using Java.

M.Health Therapy Center

A software solution designed to manage mental health clinics
efficiently. It handles patient records,
appointment scheduling, therapist management,
and therapy tracking while ensuring data security.
